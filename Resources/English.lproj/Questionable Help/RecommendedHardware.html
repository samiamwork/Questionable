Recommended Hardware Configurations

Questionable works on all Apple Computers running OS 10.4 (Tiger) and up, but some additional hardware will make the experience more authentic.

A Second Display

While it is possible to play the game with a single display, you would end up with one of two problems: the players would be able to see the answers when they buzz in (if the host and player are looking at the screen), or the players would see nothing and the host would have to read everything to them (if only the host was looking at the screen).

The best configuration is one with two displays: one that the host looks at as they face the players (usually the main display with the menubar) and a second display that shows the main game window in fullscreen to the players.

Fortunately most Macs support secondary monitors in extended desktop mode. The only Mac currently produced that does not have support for a second display is the Mac mini. Most of these Macs also support TV-out so you can use a TV as your second display in a pinch, but this isn't recommended due to the low resolution of standard definition TVs. If you have an HDTV you should be able to connect to it using either DVI (recommended) or VGA.

Support for playing with a single display is in the works.

Buzzers

Questionable supports four different devices to use as player buzzers:

  • Keyboards *
  • Mice
  • Joysticks
  • Gamepads

* Keyboards are not really recommended because they're a bit akward as buzzers, but they do indeed work. The other three (especially mice) make fantastic buzzers.

NOTE: When using keyboards and mice as buzzers, be sure to plug them in while Questionable is running. If you do not do this the system will treat them as normal mice and keyboards and may interfere with the running of the program.

Also you'll probably want to use a USB hub though since most Macs don't have more than four USB ports You're probably already using one or two of them, anyway. It is also recommended that you get a powered USB hub so that the devices don't draw too much power causing the system to turn off the ports. It all depends on the devices you use.

For more detailed information on what works and tips on getting things set up check the Controllers page.
