Interface Detail

Questionable has two main windows as pictured below: the Host Window and the Game Window.

Details on preferences are here.

1. The Host Window

From this window the host sets up and runs the game.

More info.

2. The Game Window

This is the window that the players will see. It is intended to be run fullscreen on a secondary display and will display the game once it has started. Other than setting it to fullscreen or moving it around you will not otherwise normally interact with this window.
